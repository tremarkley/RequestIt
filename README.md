# RequestIt

> Spotify Application for requesting Songs in shared spaces

## Table of Contents

1. [Usage](#Usage)
1. [Requirements](#requirements)
1. [Development](#development)

## Usage

> Steps for using:
> 1. Build the bundle.js for client side code using "npm run build"
> 3. Start the server by running "npm start"
> 4. Access server at http://127.0.0.1:${process.env.REQUEST_PORT || 3333}

## Requirements

This project is using:
- Node
- Express
- React
- Socket.io
- Webpack

## Development

### Installing Dependencies

From within the root directory:
```sh
Install dependencies: npm install
Start webpack: npm run build
Start server: npm start
```
